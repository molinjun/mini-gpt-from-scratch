# 09 ReLU Adds a Bend

## Section

Neural Network Basics

## Slide-Visible Text

- Linear layers alone stay linear
- ReLU keeps positive values
- Nonlinearity adds expressive power

## Speaker Notes

If we only stack linear layers, the result is still linear. ReLU adds a simple bend. Negative values become zero, and positive values stay positive. This gives the model expressive power, so it can fit patterns that a single straight line cannot fit.

## Visual Direction

Side-by-side plot: a straight line underfitting curved data, then a piecewise line after ReLU bends the function.
